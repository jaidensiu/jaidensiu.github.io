# [jaidensiu.vercel.app](https://jaidensiu.vercel.app)

To start development server
```
npm start
```

To deploy onto GitHub Pages and Vercel
```
npm run deploy
```

- HTML
- CSS
- JavaScript
- React.js
- GitHub Actions
- Vercel
